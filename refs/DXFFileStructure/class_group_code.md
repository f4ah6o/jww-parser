Class Group Codes in DXF Files (DXF)
====================================

The following is an example of the CLASSES section of a DXF file:

0

SECTION

2

CLASSES _Beginning of CLASSES section_
0

CLASS

1

<class dxf record>

2

<class name>

3

<app name>

90

<flag>

280

<flag>

281

<flag>_Repeats for each entry_
0

ENDSEC _End of CLASSES section_
